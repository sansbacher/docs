PowerShell for Admins - CLI Task Answers

Here's one solution to the CLI Task Test, there are different commands you could have used (eg. ls instead of dir - as ls is the Linux/UNIX command for "list", or you could have used the actual PowerShell command Get-ChildItem or its alias gci - they are all the same). There is no best answer, whatever lets you move around PowerShell and get things done is correct. Use whatever you feel comfortable with (DOS aliases have the benefit of being usable in a CMD Command Line as well - twice the knowledge/half the work!).
